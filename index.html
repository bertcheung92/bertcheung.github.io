A metroidvania platformer done in Unity where the player is the sword and enemies are used to wield you
Game Design
  • Bert Cheung
  • Quinn Macdonald - Project Manager, Visual Designer, Level Designer
  • Zach Chan - Project Manager, Programmer
  • Mark Zhang - Visual Designer, Media Specialist
  • Yi Xiang - Visual Designer, Programmer
My Role: Programmer, Media Specialist

Soulmonger is a combat based metroidvania 2D platformer using the Unity engine. The player must navigate through a series of unique areas solving combat and movement based challenges. Narrative moments and characters will be interspersed throughout the game’s progression, giving context to the game’s story and setting. Set in a dark comedy fantasy world, this side scrolling metroidvania twists the genre by letting you play as Styx, an ancient demonic sword trying to escape its dark prison. Styx is able to possess the bodies of fallen enemies to wield itself, making each fight an opportunity to gain a new set of abilities, depending on the enemy’s type. This game brings a unique spin by making enemies a resource, power-up, and challenge, all at the same time.
