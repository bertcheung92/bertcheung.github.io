Indosage Massage

A mockup site made for IAT's Web Design and Development course
  • Bert Cheung
  • Polina Chzan
My Roles: Web Coder/Designer

Process

We were asked to make a mockup for a website based on a service of our choice. The requirements were to work with HTML5 and CSS using flexboxes and grids as part of what we have learned in the course. We chose a massage service and created a stylepage and a sitemap to create a consistent and streamlined experience for site visitors. The hardest part of designing and creating the website was making the cards and animated, interacted elements as seen in the Contact Us and index pages.

Result

The Indosage Massage site turned out great! I had a fun time learning how to apply what I've learned with CSS, HTML5, grid/flexbox and styling principles to make the website. Polina was an amazing designer and I'm amazed at how well we were able to tie the site together.

Some of the skills I've used include:

  • HTML5 and CSS principles including anchors, divs, animated elements
  • Creating responsive designs for websites
  • Applying colour theory and design principles to create a sleek style for our site
  • Using flexbox and grids in CSS
